# nuna

## capslock 增强工具

为capslock做增强，增加capslock的使用率

键位映射列表

1. CAPS + A = HOME (按下Caps+ A组合键，发送HOME键，调到行首)
2. CAPS + E = END (调到行尾)
3. CAPS + H = LEFT (左移)
4. CAPS + J = UP （上移）
5. CAPS + K = DOWN （下移）
6. CAPS + L = RIGHT （右移）
7. CAPS + F  = CTRL + RIGHT （光标按单词向右调）
8. CAPS + B  = CTRL + LEFT（光标按单词向左调）
9. CAPS + SPACE  = BACKSPACE（退格键）
10. CAPS + D = DEL（删除键）
11. CAPS + Q = CTRL + A
12. CAPS + S =  CTRL + S 
13. CAPS + W =  CTRL + W 
14. CAPS + Z = CTRL + Z 
15. CAPS + X = CTRL + X
16. CAPS + C = CTRL + C
17. CAPS + V = CTRL + V
